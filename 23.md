为博客添加阅读时长估算和优雅的"回到顶部"功能。

1. 阅读时长估算：
   - 在 src/pages/blog/[slug].astro 中，根据文章内容的纯文本长度（去除 markdown 语法）估算阅读时长。
   - 计算公式：字数 / 300 (中文) 或 字数 / 200 (英文)，向上取整。
   - 在文章标题下方的元数据区域，添加一个图标 🕒 和文案 "预计阅读 x 分钟"。

2. 优雅的回到顶部按钮：
   - 创建 React 组件 src/components/ScrollToTop.tsx。
   - 渲染一个固定在页面右下角的圆形按钮 (带有向上箭头图标)。
   - 默认隐藏 (opacity-0 translate-y-4 pointer-events-none)。
   - 当页面向下滚动超过 300px 时，按钮带过渡动画浮现出来。
   - 点击时，使用 window.scrollTo({ top: 0, behavior: 'smooth' }) 平滑回到顶部。
   - 样式要求：毛玻璃背景，深色/亮色模式下都要清晰可见，hover 时发光。

3. 在 BaseLayout.astro 的底部引入 ScrollToTop 组件 (client:load)。
