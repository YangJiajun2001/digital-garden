最后一步，为博客添加全局动效，提升流畅度。

1. 页面切换过渡：
   在 astro.config.mjs 中开启 View Transitions (动画)。
   在 BaseLayout.astro 的 <head> 中引入 <ViewTransitions />。
   设置页面切换时的轻微淡入效果。

2. 滚动入场动画：
   创建 React 组件 src/components/ScrollReveal.tsx。
   - 使用 IntersectionObserver 监听子元素是否进入视口。
   - 进入视口时，应用 CSS 动画从 opacity-0 translate-y-8 过渡到 opacity-100 translate-y-0。
   - 过渡时长 0.6s，使用 ease-out 曲线。
   - 支持延迟属性，以便列表项目可以依次显现。

3. 应用动画：
   在首页的 Hero 标题、每一张 Bento Card，以及详情页的标题和正文区域，用 ScrollReveal 包裹。

请提供 ViewTransitions 的配置方法、ScrollReveal.tsx 组件代码，以及如何在页面中使用它的示例。
