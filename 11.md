现在为博客添加全站离线搜索功能，使用 Pagefind。

要求：
1. 运行 npm install pagefind 安装依赖。
2. 修改 package.json 中的 build 脚本，在 astro build 之后执行 pagefind --site dist (如果是本地开发 dev 模式则不需要)。
3. 在 src/components/SearchModal.tsx 创建一个搜索弹窗组件：
   - 包含一个搜索输入框和结果列表。
   - 按下 Cmd/Ctrl + K 快捷键唤起弹窗，按 Esc 关闭。
   - 引入 Pagefind 的 JS：/pagefind/pagefind.js。
   - 实时防抖搜索，搜索结果展示文章标题、高亮匹配的上下文片段，点击跳转。
   - 整体风格采用深色毛玻璃效果，与网站现有风格一致。
4. 在 BaseLayout.astro 的导航栏添加一个搜索按钮（带搜索图标），点击唤起 SearchModal，设置 client:load。
5. 确保在构建后，文章内容能被正确索引（在详情页的 article 标签添加 data-pagefind-body）。
