现在添加创意核心功能：交互式知识图谱页面。

1. 安装依赖：npm install react-force-graph-2d
2. 在项目根目录生成静态数据：创建一个脚本或在构建时，生成一个包含图谱数据的 JSON (例如 /graph-data.json)，数据结构为：
   { nodes: [ { id: "slug", name: "文章标题", status: "seed"|"growing"|"evergreen", tags: [...] } ], links: [ { source: "slug1", target: "slug2" } ] }
   （links 的生成逻辑：如果文章A的正文包含 [[文章B]]，则有一条从A指向B的线。如果文章共享相同标签，也可以生成一条淡色的连线）。
3. 创建 src/components/KnowledgeGraph.tsx (React 组件)：
   - 使用 react-force-graph-2d 渲染图谱。
   - 节点颜色根据 status 变化：seed绿色，growth蓝色，evergreen金色。
   - 节点大小根据连接数变化。
   - 交互：鼠标悬停节点高亮其关联连线，点击节点跳转到 /blog/[slug]。
   - 画布背景透明或极深色，连线使用半透明的渐变色。
4. 创建 src/pages/garden.astro，全屏渲染 KnowledgeGraph 组件（client:load），右上角增加一个图例说明颜色代表的状态。
