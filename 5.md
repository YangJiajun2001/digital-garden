1. 在 src/components/ 下创建 SpotlightEffect.tsx (React 组件)。
2. 功能要求：
   - 监听鼠标移动事件。
   - 在页面上渲染一个径向渐变的圆形光斑，跟随鼠标平滑移动。
   - 光斑效果：类似手电筒照亮的效果，背景是透明的，但通过 CSS mix-blend-mode 或者一个半透明的径向渐变层，让鼠标周围的内容稍微变亮或有颜色倾向 (如淡绿色光晕)。
   - 离开页面时光斑消失。
3. 将此 React 组件集成到 src/layouts/BaseLayout.astro 的 <body> 标签底部，确保它在所有页面生效，且 z-index 要合适（不阻挡点击，pointer-events-none）。
4. 优化：使用 requestAnimationFrame 实现平滑的延迟跟随，而不是生硬的同步跟随。
