为博客文章详情页添加悬浮高亮目录功能。

要求：
1. 数据提取：
   - 修改 content/config.ts 中 blog 集合的配置，如果 Astro 默认没有返回 headings，请确保在 [slug].astro 中能获取到 Markdown 渲染出的 h2, h3 标题列表（包含 depth, slug, text）。
2. 创建 src/components/FloatingToc.tsx (React 组件)：
   - 接收 headings 数组作为 props。
   - 仅当屏幕宽度大于等于 lg (1024px) 时显示，移动端隐藏。
3. 定位与样式：
   - 采用 fixed 定位，放在页面右侧留白处 (right-[calc((100vw-1280px)/2)] top-32 之类，根据你的布局微调)。
   - 宽度固定约 200px。
   - 毛玻璃背景：bg-black/40 backdrop-blur-md border border-white/5 rounded-lg p-4。
4. 高亮交互逻辑 (核心)：
   - 使用 Intersection Observer API 监听页面上所有 h2, h3 标签。
   - 当某个标题进入视口可视区域时，将其对应的目录项高亮 (文字变亮，左侧加一条绿色/青色的竖线指示器)。
   - 点击目录项，使用 document.querySelector(id).scrollIntoView({ behavior: 'smooth' }) 平滑滚动。
5. 引入：
   - 在 src/pages/blog/[slug].astro 中引入 FloatingToc，传入文章的 headings 数据 (client:load)。
