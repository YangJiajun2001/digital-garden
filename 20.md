为博客文章详情页的代码块添加一键复制按钮。

不要用 React 组件包裹，请使用 Astro 的客户端脚本 (Vanilla JS) 配合 DOM 操作实现，以保证性能和不破坏原有 Markdown 渲染。

要求：
1. 在 src/scripts/ 目录下创建 copy-code.js：
   - 监听 DOMContentLoaded 或使用 Astro 的 page-load 事件。
   - 查找所有带有 class 包含 "astro-code" 的 <pre> 标签（Astro 默认代码块容器）。
   - 在每个 <pre> 的内部右上角注入一个 <button>，设置 position: absolute，包含一个 SVG 复制图标 (📋 或简单的文字 "Copy")。
   - 点击按钮时，获取该 <pre> 内 <code> 的纯文本内容，使用 navigator.clipboard.writeText 复制。
   - 复制成功后，按钮内容变成 "Copied!" 或 ✔️ 图标，2秒后恢复原状。
2. 在 BaseLayout.astro 的底部引入此脚本：<script src="../scripts/copy-code.js"></script> (Astro 会自动打包处理)。
3. 添加必要的 CSS (在 global.css 中)：
   - 确保 <pre> 设置了 position: relative。
   - 复制按钮样式：半透明背景 (bg-gray-700/50 dark:bg-black/50)，圆角，hover 时变实心。在亮色模式和暗色模式下都要清晰可见。

请给出 copy-code.js 的完整代码、global.css 追加的样式，以及 BaseLayout.astro 的修改部分。
