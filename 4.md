现在实现博客的文章详情页。

1. 在 src/pages/blog/ 目录下创建 [slug].astro。
2. 使用 getStaticPaths 获取所有 blog 集合的路由。
3. 使用 BaseLayout 包裹内容。
4. 页面结构：
   - 顶部返回按钮（返回首页）
   - 文章标题 (大号加粗)
   - 元信息：日期、状态 Emoji、标签 (用小圆角药丸样式展示标签)
   - <article> 标签包裹 Markdown 渲染的正文内容 (使用 Astro 自带的 <Content /> 组件)。
5. 针对 Markdown 正文内容，编写 Tailwind 排版样式，要求暗色模式下阅读体验极佳：
   - 标题 h1-h6 要有明显的层级和颜色区分 (如 text-gray-100 到 text-gray-400)。
   - 段落间距舒适 (leading-relaxed, mb-6)。
   - 超链接要有高亮颜色 (如 text-emerald-400 hover:underline)。
   - 代码块要有深色背景 (bg-gray-900 rounded-lg p-4)。
   - 列表样式要清晰。
